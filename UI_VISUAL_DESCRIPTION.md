## Color Scheme

- **Primary Red**: #FF2625 (Search button, Start Workout buttons)
- **Hover Red**: #d32f2f (Darker on hover)
- **Background**: #fff (White cards)
- **Text**: Default MUI text colors
- **Chips/Badges**: Primary and secondary MUI colors
- **Skeleton**: Light gray animated gradient

## Interactions

### Hover Effects
- Cards lift up with `transform: translateY(-8px)`
- Box shadow increases to elevation 6
- Smooth transition: `all 0.3s ease`

### Click Actions
- **Search Button**: Triggers immediate search
- **Quick Start**: Sets search term and triggers search
- **Filters**: Toggles filter and updates results
- **Favorite**: Toggles heart icon (outlined â†” filled)
- **Share**: Opens native share or copies to clipboard
- **Start Workout**: Future workout initiation
- **Retry**: Reloads data on error

### Animations
- Skeleton screens pulse while loading
- Cards fade in when loaded
- Typing indicator blinks
- Filter badges update dynamically

## Accessibility Features

- **Keyboard Navigation**: Tab through all interactive elements
- **ARIA Labels**: Proper labels on all buttons and icons
- **Screen Reader**: Descriptive text for all actions
- **Color Contrast**: Meets WCAG 2.1 AA standards
- **Focus Indicators**: Visible focus states
- **Semantic HTML**: Proper heading hierarchy
